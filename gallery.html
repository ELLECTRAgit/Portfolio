Мои проекты
Мне нравится работать с различными задачами - от воплощения вашего дизайна в реальность до создания продуманного пользовательского интерфейса. Умею верстать адаптивные сайты под планшеты и мобильные устройства.

При разработке использую:

CSS3, HTML5, JavaScript, JQuery, PHP, MySQL, Wordpress, Git, Figma, Photoshop.
Описание
Один из блоков приложения Mesto, цель которого создание личного кабинета пользователя с возможностью загрузки фотографий своих путешествий для всеобщего просмотра.
Требования:
- фото пользователя должно отражаться в круге
- должна быть реализована возможность редактирования имени и рода деятельности
- должна быть реализована возможность поставить и снять лайк
- в процессе верстки необходимо использовать технологию БЭМ
Используемые технологии: HTML, CSS, БЭМ, JavaScript, Figma

Описание
Одностраничный сайт, привлекающий внимание 3D дизайном.
Требования:
- вызвать у пользователя желание изучать сайт при минимальный объеме текста

Описание
Сайт создан, чтобы показать возможности Wordpress для заказчика.
При создании сайта на Wordpress вам не потребуется держать в штате веб-разработчика на постоянной основе!
Завершенный мною проект даст возможность изменения его контента, изображений, цветовой гаммы любому сотруднику Вашей организации без наличия профессиональных навыков. Изменения производятся путем нажатия нескольких интуитивно понятных кнопок.
